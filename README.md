# Prompt Analyzer Telegram Bot

AI бот, который анализирует фотографии и создаёт профессиональные промпты.

## Deploy на Render

1. Загрузите репозиторий
2. Перейдите в Render
3. Нажмите New → Blueprint
4. Подключите репозиторий
5. Добавьте переменные окружения:

BOT_TOKEN
GEMINI_API_KEY
CHANNEL_ID

Готово.
